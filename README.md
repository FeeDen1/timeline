# Тестовое Only - React+TS+Webpack+Swiper+GSAP

# Резюме https://hh.ru/resume/3ca50924ff0f8e38bd0039ed1f4e55365a6b33

<h4>Структуру проекта я старался делать относительно того, как в теории я бы видел это встроенное в проект. Конечно, там есть вещи, которые можно было бы сделать лучше с архитектурной точки зрения, но для тестового думаю это overkill, но если что буду рад на эту тему подискутировать</h4>
<h6>Еще я намеренно не придерживался и изменил слегка мобильную версию страницы - посмотрев как она выглядит в оригинальном варианте на небольших мобильных, мне кажется с точки зрения UX больше смысла держать переключалку интервалов у самих интервалов - чтобы пользователь видел что меняется</h6>
<h6>Также я заметил в видео-демонстрации, что кнопки переключения и точки пагинации показывают одно и то же - интервалы, поэтому мне показалось лучше идеей сделать точки как пагинацию по фактам - хотелось сделать чуть лучше и вставить свои пару копеек ахах</h6>

## Стек

| Категория | Технологии |
|------------|-------------|
| Билд | Webpack (самонастроенный) |
| Фреймворк | React 19 |
| Стили | SCSS Modules + CSS Variables |
| Анимации | GSAP |
| Слайдер | Swiper |

---
## Инструкция по запуску
```bash
git clone https://github.com/FeeDen1/timeline.git
npm install
npm run build:dev || npm run build:prod Для сборки проекта
npm rum start
```
